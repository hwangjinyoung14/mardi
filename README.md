# 🛒마르디메크르디 쇼핑몰 프로젝트🛒
이 프로젝트에서는 마르디메크르디 쇼핑몰을 리액트 기반으로 진행했습니다.
<br/><br/>

## 데모 사이트
[프로젝트 데모 사이트 바로가기](https://boisterous-vacherin-57140f.netlify.app/)
<br/><br/>

## 주요기능
- 전체 상품 목록을 볼 수 있습니다.
- 로그인을 통해 인증된 사용자만 상세 페이지에 접근할 수 있습니다.
- 검색 기능을 통해 원하는 상품을 찾을 수 있습니다.
  <br/><br/>

## 사용된 기술 
- React
- React Router
- CSS스타일링
- JSON 데이터베이스
